pele.angleaxis.AATopology¶

class pele.angleaxis.AATopology(sites=[])[source]¶

Angle axis topology

An angle axis topology stores all topology information for an angle axis system. The AATopology is composed of several angle axis sites (AASiteType), which describe the shape of the angle axis site and each site carries a position and orientation. Therefore, the length of the coordinate array must be 6*number_of_sites.

Parameters :

sites : iteratable

list of sites in the system
