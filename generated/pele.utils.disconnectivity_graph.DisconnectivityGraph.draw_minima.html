pele.utils.disconnectivity_graph.DisconnectivityGraph.draw_minima¶

DisconnectivityGraph.draw_minima(minima, axes=None, **kwargs)[source]¶

draw a specified list of minima as points

minima : list of minima axes : matlplotlib axes

Draw them on these axes. If None, use self.axes.
kwargs : kwargs
additional keyword arguments will be passed to matplolib.Axes.scatter(). Use this to specify how to draw the points.
