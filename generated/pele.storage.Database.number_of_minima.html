pele.storage.Database.number_of_minima¶

Database.number_of_minima()[source]¶

return the number of minima in the database

Notes

This is much faster than len(database.minima()), but is is not instantaneous. It takes a longer time for larger databases. The first call to number_of_minima() can be much faster than subsequent calls.
